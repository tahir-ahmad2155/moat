# Moat - Redefining Performance

A high-end, ultra-modern anti-gravity scrolling website inspired by the Apple MacBook Pro experience, redesigned with an off-white luxury aesthetic.

## Features

- **Off-White Luxury System**: Minimalist color palette (#F8F6F2 primary).
- **Anti-Gravity Motion**: Continuous floating animations and soft shadows.
- **Smooth Scrolling**: Powered by Lenis for buttery smooth navigation.
- **GSAP Animations**: Advanced scroll-triggered reveals and parallax effects.
- **Responsive Design**: Fully optimized for all device sizes.

## How to Run

1. Open `index.html` in your browser.
2. For the best development experience, use a local server (e.g., Live Server in VS Code):
   - Right-click `index.html` in VS Code.
   - Select "Open with Live Server".

## Customization

- **Images**: Replace the CSS placeholder in `.mockup-placeholder` with your own product image or use the `generate_image` tool if needed.
- **Content**: Update text in `index.html`.
- **Styles**: Modify `style.css` to adjust colors, fonts, or spacing.

## Technologies Used

- HTML5
- CSS3 (Variables, Flexbox, Grid)
- JavaScript (ES6+)
- GSAP (GreenSock Animation Platform)
- Lenis (Smooth Scroll)
